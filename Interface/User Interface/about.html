Jedes Jahr werden 100.000 Menschen weltweit Opfer von Keyloggern und Angriffen durch Man-in-the-Middle Attacken. Genau deshalb haben wir xIdent entwickelt. Durch unseren open-source Proxyserver und dessen genialen Authentifikationsverfahren können wir einfach verhindern, dass ihre Daten gestohlen werden. Das ganze Projekt wurde im Rahmen von JugendHackt Süd entwickelt.
